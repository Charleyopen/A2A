# Face2Face — 讨论组

这是一个很有意思的原创项目，可以看到未来工作范式的一角。
大概意思是数字分身多个大模型。有一个初始议题，多个不同数字人（不只是大模型而是有人格的分身）讨论后，给出结论。
加上针对结论的执行，似乎脑力劳动终将走向终结。人类会定目标、监督和把关。
That's why 审美、坚持、决策力是下一阶段的人累竞争之地。

支持多个**讨论组**，每组三人围绕一个主题进行 **2 轮**讨论，人设基于真实公开言论，最后由指定成员生成记录与总结。

## 流程

1. **选择讨论组**（命令行输入 1 或 2，网页端下拉选择）  
2. **输入主题**  
3. **第 1 轮**：三人按组内顺序依次就主题发言  
4. **第 2 轮**：三人针对当前讨论再各发言一次  
5. **总结**：由该组指定成员根据完整对话生成「记录与总结」

## 环境与依赖

- Python 3.9+
- 依赖：`pip install -r requirements.txt`

## 配置

使用**智慧大模型（智谱 AI）**接口。首次使用请复制 `config.example.py` 为 `config.py` 并填入 API Key，或通过环境变量设置：

```bash
export ZHIPU_API_KEY="你的API_KEY"
```

## 运行

### 命令行

```bash
cd Face2Face
pip install -r requirements.txt
python3 main.py
```

按提示**选择讨论组（1 或 2）**、输入主题后，程序会依次输出每轮对话和最终总结，并将完整结果（含 `group_id`、`group_name`、`turns`、`summary`、`personas` 等）保存到 `outputs/主题名.json`。

也可通过参数指定讨论组（若你后续为 main.py 增加 argparse）：

- 讨论组 1：乔布斯、马斯克、张一鸣  
- 讨论组 2：巴菲特、特朗普、比尔·盖茨  

### 网页版（可视化 + 历史记录）

```bash
pip install -r requirements.txt
python3 app.py
```

浏览器打开 http://127.0.0.1:5001：先在下拉框选择**讨论组一**或**讨论组二**，输入主题并「开始讨论」；中间区域展示当前对话，右侧为**讨论历史**（会显示讨论组名称），点击某条可查看该次完整记录与总结。
